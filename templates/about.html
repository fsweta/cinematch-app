About CineMatch

CineMatch is an AI-powered movie recommendation system built as a college internship project. It uses content-based filtering with NLP techniques to suggest movies similar to ones you already love.

🤖

How it Works

We use CountVectorizer + Cosine Similarity on movie tags (genres, cast, crew, keywords, overview) to find the most similar movies.

📊

Dataset

Trained on TMDB 5000 Movies dataset from Kaggle — containing 5000+ movies with detailed metadata.

⚙️

Tech Stack

Python · Flask · Pandas · Scikit-learn · NLTK · HTML · CSS · JavaScript · OMDB API

Built for College Internship

This project demonstrates the practical application of Machine Learning and NLP in building recommendation systems.
